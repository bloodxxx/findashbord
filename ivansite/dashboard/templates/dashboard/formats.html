Поддерживаемые форматы файлов
Excel
.xlsx .xls
Таблицы Microsoft Excel. Данные должны быть на первом листе, первая строка — заголовки колонок.
CSV
.csv
Таблица с разделителями (запятая или точка с запятой). Кодировка UTF-8 или CP1251. Первая строка — заголовки.
XML
.xml
Табличная XML-выгрузка. Колонки передаются как теги или атрибуты. Структура должна быть плоской.
Ограничение: система принимает только табличные финансовые файлы с обязательными колонками. PDF, Word, договоры и произвольные текстовые файлы не поддерживаются.
